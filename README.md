어딘가 많이 아픈 바둑 (Glitchy Baduk) 🤪경고: 이 게임은 정상적인 바둑 대국을 위해 만들어지지 않았습니다. 플레이 시 시각적 테러와 극심한 스트레스를 유발할 수 있습니다.📝 소개이 프로젝트는 "어떻게 하면 사용자에게 최대한의 짜증을 유발할 수 있을까?"라는 철학 아래 개발된 파이썬 바둑 게임입니다. 정상적으로 작동하는 것 같지만, 결정적인 순간에 렉이 걸리거나 돌이 제멋대로 놓이는 최첨단 '킹받음' 엔진이 탑재되어 있습니다.✨ 주요 기능 (이라 쓰고 버그라 읽음)시각 테러: 핫핑크 배경과 형광 초록색 격자로 사용자의 눈을 공격합니다.수전증 인터페이스: 바둑판과 돌이 쉴 새 없이 떨려 정확한 착수를 방해합니다.랜덤 입력 무시: 당신의 클릭은 약 30% 확률로 무시됩니다.미끄러짐 효과: 10% 확률로 클릭한 곳 옆 칸에 돌이 놓입니다.트롤링 AI: 흑을 잡았는데 백돌이 놓이거나, 한 번에 두 번 두어지는 등 규칙을 파괴합니다.최적화 실패(의도됨): 간헐적인 프레임 드랍으로 옛날 컴퓨터의 향수를 느낄 수 있습니다.🚀 실행 방법필수 요구 사항Python 3.xPygame설치 및 실행저장소를 클론하거나 다운로드합니다.필요한 라이브러리를 설치합니다.pip install -r requirements.txt
게임을 실행합니다.python glitchy_baduk.py
